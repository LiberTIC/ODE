===========
DEVELOPMENT
===========

We ship a Makefile with a few useful targets to setup a development
environment.

Install in development mode (you probably want to do that in a `virtual environment <http://www.virtualenv.org>`_)::

    $ make develop

Run the test suite::

    $ make test
